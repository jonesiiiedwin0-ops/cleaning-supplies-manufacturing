## 3. How to Use Each MCP Tool

This section details the specific use cases and best practices for each tool.

### GitHub

*   **Purpose:** Version control, collaboration on code/documentation, task tracking (via issues).
*   **Key Uses:** Managing `EXECUTION_GUIDE.md`, tracking project progress, storing scripts.
*   **Best Practices:** Commit frequently, use clear commit messages, create branches for major changes.

### Google Sheets

*   **Purpose:** Data analysis, financial tracking, inventory management.
*   **Key Uses:** Production logs, sales forecasts, raw material tracking.
*   **Best Practices:** Use consistent formatting, protect sensitive sheets, share appropriately.

### Todoist

*   **Purpose:** Personal and team task management.
*   **Key Uses:** Daily task lists, recurring tasks, project milestones.
*   **Best Practices:** Break down large tasks, set due dates, use labels/priorities.

### Notion

*   **Purpose:** Knowledge base, documentation, project management, team wikis.
*   **Key Uses:** SOPs, meeting notes, project briefs, long-form documentation.
*   **Best Practices:** Maintain clear structure, link related pages, keep content updated.

### Airtable

*   **Purpose:** Database management, custom workflows, data visualization.
*   **Key Uses:** Supplier database, product catalog, customer relationship management.
*   **Best Practices:** Design clear tables, link records effectively, use views for different purposes.

## 4. Daily Operational Workflow

1.  **Morning Check-in:** Review Todoist tasks, check daily metrics dashboard.
2.  **Production Planning:** Verify raw material availability, schedule production runs.
3.  **Execution:** Oversee manufacturing processes.
4.  **Data Entry:** Record production output, quality control checks in Google Sheets/Airtable.
5.  **Problem Solving:** Address any immediate issues, document in Notion/GitHub issues.
6.  **End-of-Day Review:** Update task statuses, prepare for next day.

## 5. Key Metrics to Track

*   **Production Output:** Units produced per shift/day.
*   **Quality Control:** Defect rate, rework rate.
*   **Inventory Levels:** Raw materials, finished goods.
*   **Order Fulfillment:** On-time delivery rate.
*   **Cost Efficiency:** Cost per unit, waste reduction.
*   **Team Productivity:** Task completion rates.

## 6. Troubleshooting Guide

*   **Issue:** Machine malfunction.
    *   **Action:** Refer to machine-specific manual, contact maintenance.
    *   **Escalate:** If not resolved within X hours, contact Y.
*   **Issue:** Data inconsistency.
    *   **Action:** Cross-reference sources, check data entry logs.
    *   **Escalate:** If source of error unclear, contact Data Lead.

## 7. Emergency Procedures

*   **Fire:** Activate alarm, evacuate building, call emergency services.
*   **Chemical Spill:** Refer to MSDS, use appropriate PPE, contain spill.
*   **Power Outage:** Activate backup generators, safely shut down critical equipment.
